# Sistema de Finanças

## ⚠️ Modo provisório de utilização

Atualmente o sistema ainda **não está hospedado em um servidor**.
Para utilizá-lo, é necessário rodar o projeto localmente na sua máquina.

Abaixo está o passo a passo para executar o sistema.

---

## 📦 Pré-requisitos

Antes de iniciar, verifique se você possui instalado:

* **Node.js**
Link: https://nodejs.org/en/download
* **npm** (geralmente já vem com o Node)

---

## 🚀 Como executar o projeto

1. **Baixe ou clone** a pasta do projeto para seu computador.

2. **Abra a pasta do projeto em uma IDE**
   (recomendado: WebStorm, VS Code ou similar).

3. Localize o arquivo **`package.json`**.

4. Clique com o **botão direito** no arquivo `package.json`.

5. Clique em:

```
Open in → Terminal
```

6. Quando o terminal abrir, execute o comando:

```bash
npm start
```

7. Após executar o comando, o sistema iniciará um **servidor local**.

8. O terminal mostrará um **link de acesso** (exemplo):

```
http://localhost:3000
```

9. **Clique no link ou copie para o navegador** para começar a usar o sistema.

---

## 🧪 Observação

Este é apenas um **modo provisório de execução**.
Em breve o sistema será **hospedado em um servidor**, permitindo acesso direto pelo navegador sem necessidade de instalar ou executar localmente.
